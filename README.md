## 使用说明
终端运行main.py后，输入读取Url的文件名，以及保存结果的文件名。

```
# 运行后会提示输入两个文件名
$ python3 main.py 

# filename1 是获取url的文件  filename2是保存结果的文件
# option 只支持 lsdj gwd
$ python3 commandline.py option filename1 filename2
```
